# Fleet Tracking Dashboard

This project is built using [Vite](https://vitejs.dev/), a fast and modern build tool for web applications.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload automatically if you make edits.\
You will also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include hashes for long-term caching.

### `npm run preview`

Runs the locally built app to verify the production build before deploying.
